# Settings Reference

## GenerationSettings

All settings are configured through the `GenerationSettings` class or the importer window UI.

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `ClassName` | string | null | Entry class name (overridden by `#class:` directive) |
| `DatabaseName` | string | null | Database class name (overridden by `#database:` directive) |
| `NamespaceName` | string | null | C# namespace (overridden by `#namespace:` directive) |
| `ScriptOutputPath` | string | `Assets/Scripts/Generated/` | Directory for generated C# files |
| `AssetOutputPath` | string | `Assets/Data/` | Directory for generated asset files |
| `AssetPrefix` | string | `""` | Prefix for asset file names |
| `UseListInsteadOfArray` | bool | true | Use `List<T>` instead of `T[]` for array types |
| `GenerateDatabaseContainer` | bool | true | Generate a database class with entries list |
| `GenerateEnumScripts` | bool | false | Generate separate files for enum types |
| `OverwriteExistingAssets` | bool | true | Overwrite existing asset files on re-import |
| `DeleteOrphanedAssets` | bool | false | Delete assets that no longer have source data |
| `SanitizeFieldNames` | bool | true | Convert headers to valid C# identifiers |
| `UseSerializeField` | bool | false | Use `[SerializeField] private` instead of `public` |
| `GeneratePropertyAccessors` | bool | false | Generate read-only properties (requires UseSerializeField) |
| `GenerateTooltips` | bool | false | Add `[Tooltip]` attributes to all fields |
| `GenerateCreateAssetMenu` | bool | false | Add `[CreateAssetMenu]` to generated classes |
| `SerializableClassMode` | bool | false | Generate `[Serializable]` class instead of ScriptableObject |
| `AddAutoGeneratedHeader` | bool | true | Add auto-generated comment header to files |
| `AssetNamingColumn` | string | null | Column to use for asset file names |
| `ArrayDelimiter` | string | `;` | Delimiter for array values in CSV |
| `NullToken` | string | `null` | String that represents null/empty values |
| `CommentPrefix` | string | `#` | Line prefix for comments in CSV |
| `HeaderRowCount` | int | 3 | Number of header rows (1, 2, or 3) |

## EditorPrefs Persistence

The importer window persists the following settings between sessions using `EditorPrefs`:

| Key | Setting |
|-----|---------|
| `D2SO_ScriptOutputPath` | Script output directory |
| `D2SO_AssetOutputPath` | Asset output directory |
| `D2SO_Namespace` | Default namespace |
| `D2SO_GenerateTooltips` | Tooltip generation toggle |
| `D2SO_SanitizeFieldNames` | Field name sanitization toggle |
| `D2SO_UseList` | List vs array preference |
